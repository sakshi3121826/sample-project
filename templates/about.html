About this project

This application demonstrates a full Flask frontend and backend for ingesting and normalizing data from many formats.

  • Video: Azure Video Indexer / Video Analyzer
  • Text: regex parsing, OpenAI normalization
  • Excel: Pandas ingestion and sheet normalization
  • Transactions: unified JSON schema

Use this project as a foundation to connect real Azure services and full video analytics pipelines.
